
##我的进步阶梯（博客）


想尝试下jekyll搭建博客的过程，然后就有了这个博客，基于**github | jekyll | bootstrap | markdown** 这样的方式真的很方便。 
只是方便自己的学习总结和管理，难免有错误，从不断的总结中去学习和提高自己，所以算是我的进步阶梯

+ 测试总结（主要工作）
+ 测试自动化的一些东西(selenium, robotfrawork，QTP)
+ 敏捷测试(关注点和方向）
